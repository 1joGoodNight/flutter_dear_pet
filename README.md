Flutter Dear Pet은 반려동물 용품을 관리하고 구매할 수 있는 Flutter 애플리케이션입니다.  
다양한 반려동물 용품 확인, 상세 정보 보기, 장바구니 담기, 구매까지 가능!

---

## 주요 기능
- **상품 목록 보기**: 그리드로 상품 확인
- **상품 상세 정보**: 클릭 시 상세 정보 확인
- **장바구니 기능**: 원하는 상품 추가
- **상품 추가**: 새로운 상품 등록 가능

---

## 시작하기
### 필수 조건
- Flutter SDK
- Android Studio or VS Code
- 모바일 디바이스 or 에뮬레이터

### 설치 방법
```bash
git clone [저장소 URL]
cd flutter_dear_pet
flutter pub get
flutter run
```

## 프로젝트 구조
```
flutter_dear_pet/
├── lib/
│   ├── main.dart        # 앱 시작점
│   ├── list_page.dart   # 상품 목록

│   ├── models/          # 데이터 모델
│   └── data/            # JSON 데이터
```
